# Scoreboard #
Good ScoreBorad For Fivem Servers


## Base ##
[ess](https://github.com/mitlight/essentialmode) (essentialmode) &
[esx](https://github.com/mitlight/es_extended) (es_extended).


## Programming languages used ##
Input:
- `HTML`
- `CSS`
- `JS`
- `LUA`


## Features ##
- Custom colors.
- Show Time.
- Show discord link.
- Online playrs & job
- Robberys


## Installation ##
in server.cfg
```
start Scoreboard
```
